🔍 AthleticsNet Scraper Bookmarklet

Automatically extract meet results from AthleticsNet and copy them to your clipboard in CSV format for easy importing.

1

Install Bookmarklet

Copy this bookmarklet code and save it as a browser bookmark:

javascript:(function(){console.log("--- SCRAPING ATHLETICSNET ---");const tables=document.querySelectorAll('table');let csvRows=["event_name,place,grade,athlete_name,mark,school,heat,gender"];let if(headerText.includes("men")||headerText.includes("boy")||headerText.includes("male")){gender="Boys";break;}}}let eventHeader=table.closest('.event-results, .m-event-results')?table.closest('.event-results, .m-event-results').querySelector('h3, h4'):table.previousElementSibling;if(!eventHeader||!['H3','H4'].includes(eventHeader.tagName)){for(let i=allHeaders.length-1;i>=0;i--){if(allHeaders[i].compareDocumentPosition(table)&Node.DOCUMENT_POSITION_FOLLOWING){eventHeader=allHeaders[i];break;}}}let rawEventName=eventHeader?eventHeader.innerText.trim().replace(/\n/g,' ').replace(/\s+/g,' '):"Unknown Event";let eventName=rawEventName.replace(/\s-\s\d{2}"(?:\s?\/\s?\d\.\d{3}m)?/,"").replace(/\s-\s\d+(?:\.\d+)?\s?(?:kg|lb)\b/i,"").replace(/\b(\d)x(\d{2,4})\sRelay\b/i,(match,legs,dist)=>`${legs}x${dist}m Relay`).replace(/\s{2,}/g," ").trim();const headers=Array.from(table.querySelectorAll('thead th')).map(th=>th.innerText.trim().toLowerCase());const placeIdx=headers.findIndex(h=>h.includes('pl'));const nameIdx=headers.findIndex(h=>h.includes('athlete')||h.includes('name'));const gradeIdx=headers.findIndex(h=>h==='yr'||h==='year'||h==='gr'||h==='grade');const markIdx=headers.findIndex(h=>h.includes('time')||h.includes('mark')||h.includes('result'));const a.athlete-name'));if(relayAthleteLinks.length>0){name=relayAthleteLinks.map(a=>a.innerText.trim()).filter(n=>n!=="").join(', ');}else{name=row.querySelector('.school, .team')?.innerText.trim()||name;}school=school.split(' - ')[0].trim();grade="";}if(name&&mark&&!isNaN(parseInt(place))){const   3. Name it "Scrape AthleticsNet"
  4. Paste code in the URL field (should start with javascript:)
  5. Save the bookmark
2

How to Use

  1. Go to AthleticsNet Meet Results Page - Navigate to the meet you want to scrape
  2. Click the "Results" Tab - Make sure you can see the results tables
  3. Click Your Bookmarklet - Click the "Scrape AthleticsNet" bookmark
  4. Results Copied - You'll see a confirmation alert with the number of results extracted
  5. Paste in CSV Import - Go to the CSV Import page and paste the data
  6. Review & Import - Verify matches and click Import
3

What It Does

  • ✅ Auto-detects gender from page headers
  • ✅ Removes hurdle heights and throwing weights from event names
  • ✅ Normalizes relay event names
  • ✅ Extracts placements, times/marks, grades, schools, heats
  • ✅ Handles relay teams correctly
  • ✅ Copies results as CSV to clipboard
  • ✅ Works with different AthleticsNet table layouts
4

Troubleshooting

❌ "Found 0 results"

  • Verify you're on the Results tab (not Meet Info)
  • Make sure results tables are visible
  • Try refreshing the page and running again

❌ Bookmarklet not working

  • Ensure the bookmark URL starts with javascript:
  • Try reinstalling the bookmarklet with fresh code
  • Test on a different meet page

❌ Unusual results or missing data

  • Some AthleticsNet layouts may have non-standard table structures
  • After pasting in CSV Import, review the preview table to see which events matched
  • You can manually adjust or exclude problematic events before importing
5

Next Steps

After scraping results:

  5. Review event matches and select which events to import
